READ ME
Create a .env file in the root directory of your project with the following environment variables:

makefile
Copy code
PORT=3000
AWS_ACCESS_KEY_ID=your-access-key-id
AWS_SECRET_ACCESS_KEY=your-secret-access-key
AWS_REGION=your-aws-region
DYNAMODB_TABLE=your-dynamodb-table-name
Replace your-access-key-id, your-secret-access-key, your-aws-region, and your-dynamodb-table-name with your actual AWS credentials and DynamoDB table name.
